 # 林口高爾夫球場雲端管理系統 API 
 ## 林口高爾夫球場雲端管理系統的後端 API 規格文件。

## 主要功能
- 使用者認證與授權
- 會員資料管理
- 公告系統
- 優惠券管理
- 報表與分析
 
<br/>

### 安裝初始軟體包並套用 Prettier(Installing initial packages and applying Pretier)

```sh
npm install && npm run format
```

<br/>

### Swagger Doc

- http://localhost:3000/docs-yaml/

<br/>

### local running

```sh
yarn run serve:offline
```

### deploy

```sh
yarn deploy
```
